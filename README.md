# MediAssist - AI Diabetes Predictor
A healthcare web app that predicts diabetes risk using a trained ML model and AWS Bedrock (Amazon Nova) for AI explanations.

## Tech Stack
- Python + Scikit-learn (ML model)
- Flask (Backend)
- AWS Bedrock - Amazon Nova (LLM)
- HTML/CSS (Frontend)

## 5 Inputs
1. Age
2. Glucose Level
3. BMI
4. Blood Pressure
5. Insulin Level
